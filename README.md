# نظام متابعة أداء الموظفين - Performance Tracking System

هذا المشروع هو نظام Python لتتبع أهداف الموظفين والمهام الخاصة بهم، مع إمكانية عرض التقدم باستخدام رسوم بيانية.

## الميزات
- إدارة الموظفين والأقسام
- إضافة أهداف لكل موظف
- إضافة مهام لكل هدف وتتبع حالة الإنجاز
- حساب نسبة اكتمال كل هدف
- رسم رسوم بيانية تقدمية لجميع الموظفين باستخدام Matplotlib
- دعم اللغة العربية بشكل صحيح مع **arabic_reshaper** و **bidi**
